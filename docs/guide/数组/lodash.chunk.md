### `_.chunk(array, [size=1])`

将数组（array）拆分成多个 `size` 长度的区块，并将这些区块组成一个新数组。 如果`array` 无法被分割成全部等长的区块，那么最后剩余的元素将组成一个区块。

#### 引入版本

1.0.1

#### 参数

1.  `array` _(Array)_: 需要处理的数组
2.  `[size=1]` _(number)_: 每个数组区块的长度

#### 返回

_(Array)_: 返回一个包含拆分区块的新数组（注：相当于一个二维数组）。

#### 例子

```js
_.chunk(['a', 'b', 'c', 'd'], 2);
// => [['a', 'b'], ['c', 'd']]
 
_.chunk(['a', 'b', 'c', 'd'], 3);
// => [['a', 'b', 'c'], ['d']]

```
