### `_.multiply(multiplier, multiplicand)`[​](#_multiplymultiplier-multiplicand "_multiplymultiplier-multiplicand的直接链接")

两个数相乘。

#### 添加版本

4.7.0

#### 参数

1.  `augend` _(number)_: 相乘的第一个数。
2.  `addend` _(number)_: 相乘的第二个数。

#### 返回

_(number)_: 返回乘积。

#### 例子

```js
_.multiply(6, 4);
// => 24

```
