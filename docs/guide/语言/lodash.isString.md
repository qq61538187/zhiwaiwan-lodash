### `_.isString(value)`[​](#_isstringvalue "_isstringvalue的直接链接")

检查 `value` 是否是原始字符串`String`或者对象。

#### 添加版本

0.1.0

#### 参数

1.  `value` _(\*)_: 要检查的值。

#### 返回

_(boolean)_: 如果 `value` 为一个字符串，那么返回 `true`，否则返回 `false`。

#### 例子

```js
_.isString('abc');
// => true
 
_.isString(1);
// => false

```
