### `_.nthArg([n=0])`[​](#_nthargn0 "_nthargn0的直接链接")

创建一个函数，这个函数返回第 `n` 个参数。如果 `n`为负数，则返回从结尾开始的第n个参数。

#### 添加版本

4.0.0

#### 参数

1.  `[n=0]` _(number)_: 要返回参数的索引值。

#### 返回

_(Function)_: 返回新的函数。

#### 例子

```js
var func = _.nthArg(1);
func('a', 'b', 'c', 'd');
// => 'b'
 
var func = _.nthArg(-2);
func('a', 'b', 'c', 'd');
// => 'c'

```
